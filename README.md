# Mental Rotation Arena

> A 3D cognitive science game built with Three.js — test and train your spatial reasoning skills.

## Overview

Mental Rotation Arena challenges players to mentally rotate 3D objects and match them to target orientations. Based on classic mental rotation psychology experiments (Shepard & Metzler, 1971).

## Features

- 🧊 **3D Objects** — Interactive Three.js rendered shapes
- 🧠 **Cognitive Training** — Scientifically-backed spatial reasoning challenges
- ⏱️ **Timed Challenges** — Score based on accuracy and response time
- 📈 **Progress Tracking** — Session statistics and improvement metrics

## Tech Stack

- **3D Engine:** Three.js
- **Language:** JavaScript (ES6+)
- **Styling:** CSS3

## Getting Started

```bash
# Just open in browser — no build step required
open index.html
```

Or serve locally:

```bash
npx serve .
# Open http://localhost:3000
```
